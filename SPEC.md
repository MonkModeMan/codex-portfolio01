# 1. PM/PMO向け「議事録・課題管理アプリ」

### 概要

会議の議事録、決定事項、ToDo、課題、担当者、期限を管理できるWebアプリ。

### 作る理由

LinkedInで発信している**PM・PMO領域**と相性が非常に良いです。

単なるTodoアプリよりも、実務利用イメージが明確で、ポートフォリオとして説得力があります。

### 主な機能

- 会議一覧
- 議事録登録
- 決定事項登録
- ToDo登録
- 課題管理
- 担当者・期限・ステータス管理
- Markdown形式で議事録出力
- CSVエクスポート

### 技術構成例

```
Frontend: React / Next.js
Backend: Next.js API Routes / Express
Database: Supabase / PostgreSQL
Auth: Supabase Auth
Deploy: Vercel
```

### Codexで学べること

- CRUD実装
- DB設計
- 認証
- フォーム設計
- 実務系UI
- GitHubで見せやすいREADME作成

### ポートフォリオでの見せ方

```
PM/PMO業務における会議後の情報整理を効率化するための議事録・課題管理アプリ。
議事録、決定事項、ToDo、課題を一元管理し、プロジェクト進行の透明性を高めることを目的とする。
```
